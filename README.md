# 📝 Editor de Texto em C#

Este projeto é um **Editor de Texto simples desenvolvido em C# (Console Application)**, com o objetivo de praticar conceitos fundamentais da linguagem, como manipulação de arquivos e a entrada e saída de dados.

O programa permite **abrir, criar, salvar e apagar arquivos de texto** diretamente pelo terminal.

---

## 🚀 Funcionalidades

✅ Abrir arquivos de texto existentes  
✅ Criar e editar arquivos de texto  
✅ Salvar arquivos em um caminho definido pelo usuário  
✅ Apagar arquivos do sistema  
✅ Interface simples via console  
✅ Navegação por menu interativo  

---

## 🛠️ Tecnologias Utilizadas

- **C#**
- **.NET**
- **Console Application**
- **Visual Studio Code**

---

## 📚 Conceitos praticados no projeto

- Estrutura de um programa em C#
- Métodos e organização de código
- Laços de repetição
- Uso de `switch-case`
- Leitura e escrita de arquivos (`StreamReader` e `StreamWriter`)
- Uso do `using` para gerenciamento de recursos
- Exclusão de arquivos com `File.Delete`
- Entrada de dados com `Console.ReadLine`
- Manipulação de strings
- Organização de menus no console
